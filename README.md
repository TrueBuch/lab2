### Name
Сущность `Name` хранит данные о полном имени человека:

- **FirstName** (string) — личное имя.
- **SurName** (string) — фамилия.
- **LastName** (string) — отчество.

**Конструкторы:**
- Только личное имя:
```
var name = new Name("Лев");
```
- Личное имя и фамилия:
```
var name = new Name("Сергей", "Пушкин");
```
- Полное имя (имя, фамилия, отчество):
```
var name = new Name("Сергей", "Пушкин", "Сергеевич");
```

### Person
Сущность `Person` хранит данные о человеке:

- **Name** (Name) — объект с именем человека.
- **Height** (int) — рост в сантиметрах.
- **Father** (Person) — ссылка на отца (может быть `null`).

**Конструкторы:**
- Имя в виде строки и рост:
```
var p = new Person("Лев", 170);
```
- Имя в виде строки, рост и отец:
```
var p = new Person("Александр", 167, father);
```
- Имя в виде объекта `Name` и рост:
```
var p = new Person(new Name("Сергей", "Пушкин"), 168);
```
- Имя в виде объекта `Name`, рост и отец:
```
var p = new Person(new Name("Сергей", "Пушкин"), 168, father);
```

### City
Сущность `City` хранит данные о городе:

- **Name** (string) — название города.
- **Routes** (Dictionary<City, int>) — маршруты к другим городам с указанием стоимости.

**Конструкторы:**
```
var city = new City("A");
```

**Методы:**
- `AddRoute(City destination, int cost, bool isReverse = false)` — добавляет маршрут к другому городу.  
  - `isReverse = true` добавляет обратный маршрут автоматически.


### Cat
Сущность `Cat` хранит данные о коте:

- **Name** (string) — имя кота.

**Конструкторы:**
```
var cat = new Cat("Барсик");
```

**Методы:**
- `ToString()` — `"Кот: Имя"`.
- `May()` — кот издаёт `"Мяу!"`.
- `May(int count)` — кот мяукает `count` раз:
```
Барсик: Мяу Мяу Мяу!
```
