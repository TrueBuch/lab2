# ЛАБОРАТОРНАЯ РАБОТА №2
**Вариант 3**  

Программа состоит из файлов: `Program.cs`, `Solution.cs`, `Name.cs`, `Person.cs`, `City.cs`, `Cat.cs`.

**Name**

Сущность имя хранит в себе следующие данные: Имя, Фамилия, Отчество
  • Имя можно создать указав только Личное имя 
  • Имя можно создать указав  Личное имя и Фамилию. 
  • Имя можно создать указав все три параметра: Личное имя, Фамилию, Отчество.

**Person**

Сущность человек хранит в себе следующие данные: Сущность Имя, Рост, Отца - ссылку на другую сущность Person
  • Человека можно создать с указанием имени в виде строки и роста в виде целого числа. 
  • Человека можно создать с указанием имени в виде строки, роста в виде целого числа и отца в виде Человека
  • Человека можно создать с указанием имени в виде объекта типа Имя и роста в виде целого числа.  
  • Человека можно создать с указанием имени в виде объекта типа Имя, роста в виде целого числаи отца в виде Человека.

**City**

Сущность город хранит в себе следующие данные: Имя, Словарь вида <City, Int> - где ключ это ссылка на другую сущность типа город, значение - стоимость проезда.
  • Город можно создать с указанием имени в виде строки.

**Cat**

Сущность кот хранит в себе следующие данные: Имя, 
  • Для создания необходимо указать имя кота. 
  • Может быть приведен к текстовой форме вида: “кот: Имя” 
  • Может помяукать, что приводит к выводу на экран следующего текста: “Имя: мяу!”, вызвать мяуканье можно без параметров. 
  • Может помяукать Nраз, что приводит к выводу на экран следующего текста: “Имя: мяу
мяу-…-мяу!”, где количество “мяу” равно N.
